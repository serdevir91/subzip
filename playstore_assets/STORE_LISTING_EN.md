# SubZip Play Store Listing (EN)

## App Details

### Short description (80 chars max)
Manage files, zip and unzip folders, and convert documents faster with SubZip.

### Full description
SubZip is a fast and practical file manager and archive utility for Android.

With SubZip, you can:
- Browse files and folders from one clean interface
- Copy, move, rename, and delete files
- Create ZIP archives from selected files and folders
- Extract ZIP files to your target directory
- Track background tasks with live progress updates and notifications
- Add frequently used folders to Favorites for quick access
- Convert supported file types:
  - Image -> PDF
  - DOCX -> PDF
  - PPTX -> PDF
  - PDF -> DOCX

Why SubZip:
- Fast file operations for daily use
- Practical ZIP tools for large files
- Device-based workflow that keeps you in control of your files

Permissions:
- Storage permissions: Required to access and manage files you choose.
- Notification permission: Used for task progress and completion updates.
- Package install permission: Used only when you explicitly open an APK file.

SubZip helps you complete file management and archive tasks in one place.
